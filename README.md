## ğŸ¨ Customization

### Mengganti Judul Film
Edit di setiap file HTML, cari dan ganti:
```html
<title>Antara Waktu, Lampu dan Mimpi</title>
```

### Mengganti Foto Aktor
Edit placeholder images di `index.html` dan `cast.html`:
```html
<img data-src="path/to/your/image.jpg" class="lazy-load">
```

### Mengganti Foto Scene
Edit placeholder images di `index.html` dan `gallery.html`:
```html
<img data-src="path/to/your/scene.jpg" class="lazy-load">
```

### Mengganti Video
Edit video source di `trailer.html` dan `watch.html`:
```html
<source src="path/to/your/video.mp4" type="video/mp4">
```
